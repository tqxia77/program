## 数据库配置

1. 复制配置文件：
```bash
cd server
cp .env.example .env
```

2. 修改 `.env`：
```env
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=你的密码
DB_DATABASE=yinling

JWT_SECRET=你的jwt密钥
JWT_EXPIRES_IN=7d
JWT_REFRESH_EXPIRES_IN=30d

# 微信小程序 AppID 和 AppSecret
WECHAT_APPID=wx_your_appid
WECHAT_SECRET=your_appsecret

# 阿里云OSS（可选）
OSS_ACCESS_KEY_ID=your_key
OSS_ACCESS_KEY_SECRET=your_secret
OSS_BUCKET=your_bucket
OSS_REGION=oss-cn-hangzhou
```

## 安装和运行

```bash
cd server

# 安装依赖
pnpm install

# 开发模式
pnpm start:dev

# 生产构建
pnpm build
pnpm start:prod
```

## API 测试

启动服务后可以用 curl 测试：

```bash
# 测试健康检查
curl http://localhost:3000/api/health

# 测试登录
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"code":"test_code","nickname":"测试用户","avatar":"","role":"elder"}'

# 测试文件上传
curl -X POST http://localhost:3000/api/upload \
  -F "file=@test.jpg"
```

## 注意事项

1. **不要修改公共模块**：`common/` 下的文件是公共代码
2. **保持接口一致**：按照文档中的接口定义实现
3. **统一响应格式**：使用 `success()` 或 `fail()` 返回
4. **需要认证的接口**：使用 `@UseGuards(JwtAuthGuard)`
5. **需要管理员权限**：使用 `@UseGuards(AdminGuard)`
