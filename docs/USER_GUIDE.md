# 银龄乐圈 - 适老化社区小程序使用说明

## 一、项目概述

银龄乐圈是一款专为回迁社区老年人设计的微信小程序，旨在帮助老年居民发现精彩社区活动、与邻里互动交流。应用采用大字体、高对比度的适老化设计，支持默认字体和大字体模式切换，让不同视力条件的老年用户都能舒适使用。

## 二、功能清单

### 2.1 活动中心（首页）

| 功能模块 | 功能说明 |
|---------|---------|
| 页面标题 | 显示"活动中心"及副标题"发现精彩社区活动" |
| 时光照相馆 | 功能卡片入口，点击提示"AI老照片修复功能正在开发中" |
| 分类筛选 | 横向滚动筛选栏，支持：全部 / 文体娱乐 / 健康养生 / 社区服务 |
| 活动列表 | 展示活动卡片，包含：封面图、标题、时间、地点 |

| 活动详情 | 点击活动卡片进入详情页，查看完整活动信息 |

### 2.2 邻里圈

| 功能模块 | 功能说明 |
|---------|---------|

### 2.3 个人中心

## 三、操作流程

### 3.1 活动报名流程

```
1. 打开"活动中心"页面
2. 浏览活动列表或使用分类筛选
3. 点击感兴趣的活动卡片
4. 进入活动详情页，查看详细信息
5. 点击底部"立即报名"按钮
6. 确认报名弹窗，点击"确定"
7. 显示"报名成功"提示
8. 返回活动列表，该活动显示"已报名"状态
```

### 3.2 邻里圈互动流程

**点赞操作**：
```
1. 进入"邻里圈"页面
2. 浏览动态列表
3. 点击动态下方的爱心图标
4. 爱心变红，点赞数+1
5. 再次点击取消点赞
```

**评论操作**：
```
1. 进入"邻里圈"页面
2. 找到想要评论的动态
3. 点击评论图标（气泡）
4. 评论区域展开
5. 在输入框中输入评论内容
6. 点击"发送"按钮
7. 评论显示在评论区
```

**发布动态**：
```
1. 进入"邻里圈"页面
2. 点击右下角"+"按钮
3. 进入发布页面
4. 输入文字内容
5. 点击"选择图片"添加照片（可选）
6. 点击"发布"按钮
7. 提示"发布成功"
8. 返回邻里圈，新动态显示在列表中
```

### 3.3 大字体模式切换

```
1. 进入"我的"页面
2. 找到"大字体模式"按钮
3. 点击按钮
4. 弹出"已切换为大字体模式"提示
5. 页面字号自动放大
6. 再次点击恢复默认模式
```

### 3.4 取消报名流程

```
1. 进入"我的"页面
2. 找到"我的报名"区域
3. 点击要取消的活动
4. 弹出确认对话框
5. 点击"确定取消"
6. 显示"已取消报名"提示
7. 该活动从列表中移除
```

### 3.5 活动管理流程（管理员）

**进入管理页面**：
```
1. 进入"我的"页面
2. 点击"活动管理"菜单项
3. 进入活动管理页面
```

**发布新活动**：
```
1. 进入活动管理页面
2. 点击右上角"发布活动"按钮
3. 填写活动信息：
   - 活动标题（必填）
   - 活动分类（必填，文体娱乐/健康养生/社区服务）
   - 活动封面图（必填，从预设图片选择）
   - 活动日期（必填）
   - 活动时间（必填）
   - 活动地点（必填）
   - 人数上限（必填）
   - 活动描述（必填）
4. 点击"发布活动"按钮
5. 提示"发布成功"
6. 返回活动列表，新活动显示在顶部
```

**编辑活动**：
```
1. 进入活动管理页面
2. 找到要编辑的活动卡片
3. 点击"编辑"按钮
4. 修改活动信息
5. 点击"保存修改"按钮
6. 提示"修改成功"
```

**删除活动**：
```
1. 进入活动管理页面
2. 找到要删除的活动卡片
3. 点击"删除"按钮
4. 弹出确认对话框
5. 点击"删除"
6. 活动从列表中移除
```

## 四、技术实现思路

### 4.1 技术栈选型

| 技术领域 | 技术选型 | 说明 |
|---------|---------|------|
| 前端框架 | Taro 4.x + React 18 | 跨端支持微信/抖音/H5 |
| 样式方案 | TailwindCSS 4 + weapp-tailwindcss | 原子化CSS，适配多端 |
| 状态管理 | React Context + LocalStorage | 字体模式全局状态 |
| 图标库 | lucide-react-taro | SVG图标自动转DataURL |
| 数据存储 | LocalStorage | 报名/点赞/评论本地持久化 |
| 组件库 | @/components/ui (shadcn/ui风格) | 统一UI组件 |

### 4.2 目录结构

```
src/
├── pages/                    # 页面组件
│   ├── index/               # 活动中心（首页）
│   ├── neighborhood/        # 邻里圈
│   ├── profile/             # 个人中心
│   ├── activity-detail/     # 活动详情

│   ├── edit-profile/        # 编辑资料
│   └── child-binding/       # 子女绑定
├── components/              # 公共组件
│   ├── ui/                  # UI组件库
│   └── safe-image.tsx       # 安全图片组件
├── store/                    # 状态管理

```

### 4.3 字体模式实现

**CSS变量定义**（`src/app.css`）：

```css
/* 默认模式 */
.font-mode-normal {
  --text-xs: 18px;
  --text-sm: 20px;
  --text-base: 24px;
  --text-lg: 28px;
  --text-xl: 32px;
  --text-2xl: 36px;
  --text-3xl: 40px;
}

/* 大字体模式 */
.font-mode-large {
  --text-xs: 30px;
  --text-sm: 32px;
  --text-base: 36px;
  --text-lg: 40px;
  --text-xl: 44px;
  --text-2xl: 48px;
  --text-3xl: 52px;
}
```

**Context管理**（`src/store/font-mode.tsx`）：

```typescript
interface FontModeContextType {
  fontMode: 'normal' | 'large'
  toggleFontMode: () => void
  isLargeMode: () => boolean
}

// Provider通过LocalStorage持久化模式状态
// 切换时触发事件通知所有页面刷新
```

**页面应用**：

```tsx
const { fontMode } = useFontMode()
const fontModeClass = fontMode === 'large' ? 'font-mode-large' : 'font-mode-normal'

return (
  <View className={`min-h-screen bg-background ${fontModeClass}`}>
    {/* 页面内容 */}
  </View>
)
```

### 4.4 图片加载处理

使用`SafeImage`组件统一处理图片加载，避免H5端报错：

```tsx
export function SafeImage({ src, className, mode = 'aspectFill', onClick }) {
  const handleError = () => {
    // 静默处理加载失败
  }

  return (
    <Image
      src={src}
      className={className}
      mode={mode}
      onError={handleError}
      onClick={onClick}
    />
  )
}
```

### 4.5 数据持久化

使用微信小程序`Taro.getStorageSync`/`Taro.setStorageSync`实现本地数据持久化：

| 数据项 | Storage Key | 数据类型 |
|-------|------------|---------|
| 已报名活动ID列表 | `signedActivityIds` | string[] |
| 用户头像 | `userAvatar` | string |
| 用户名 | `userName` | string |
| 点赞动态ID列表 | `likedPostIds` | string[] |
| 动态评论 | `postComments` | Record<string, Comment[]> |
| 字体模式 | `fontMode` | 'normal' | 'large' |

### 4.6 适老化设计要点

| 设计要素 | 实现方案 |
|---------|---------|
| 字号 | 默认18-40px，大字体30-52px，使用CSS变量统一管理 |
| 颜色对比 | 前景/背景对比度≥4.5:1，重要信息≥7:1 |
| 点击区域 | 最小44x44px，确保可触摸 |
| 圆角 | 卡片12px，按钮8px，减少锐利感 |
| 阴影 | 柔和阴影增加层次感 |
| 反馈 | 点击有视觉反馈（active状态） |
| 图标 | 配文字说明，避免纯图标交互 |
